# TooManyProfessions

Mod for Cataclysm: Bright Nights that adds several professions, one scenario so far, and a multitude of references contained within; hopefully, all decently-written.

## Professions

- Army Officer
- Bartender
- Feral Office Drone
- Feral Reporter
- Gangbanger
- Gnome Enthusiast
- Heister
- Naval Officer
- Office Drone
- Reporter

## Scenarios

- Mutant (Experiment but unlocked)
